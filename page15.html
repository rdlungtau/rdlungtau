PROJECT INTRODUCTION

This project is mainly aims to maintain the ongoing system for Fast Food Restaurant in proper  systematic way, This particular project deals with the problems on managing a Restaurant and  avoids the problems which occur when carried manually.
This software aims at automating the existing manual system and providing secure and quicker
communication within a Restaurant. This system is developed with a view to accomplished the
following points in order to enhance the effectiveness of the current management systems.
 All the available items and price can be retrieve at any instances.
 The software provides user friendly interface which provides the user to manipulate the
data with just one click.
 The system is capable of generating the bills

OBJECTIVES & SCOPE THE PROJECT

The main objective in developing this software is to automate the existing manual system and
providing secure and faster communications within a Fast Food Restaurant.
This project is useful for the better understanding of the consumer behavior. The project
encompasses the various behavior of consumer, their pre and post behavior when they eat. There
is also overview of the customer’s attraction towards fast food restaurant, mostly consumer
perceptions are changing and most of the consumer now prefer to eat in fast food restaurant. The
project work involves specific guideline for the restaurant and helps the restaurant to understand
better the consumer behavior in fast food restaurant
